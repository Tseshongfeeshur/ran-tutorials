# 基础文件操作

## 概念

- **路径**：一个文件在硬盘上的位置，由盘符、文件夹、文件（夹）名组成。如 `C:\Users\Desktop\file.txt` 就是一个路径，它表面在 C 盘中有文件夹 `Users`，其中有文件夹 `Desktop`，其中有一个文件 `files.txt`，这个路径就指向文件 `files.txt`。
  :::tip
  `C:\Users\` 是一个特殊文件夹，叫做用户文件夹，里面放置了当前用户的一些信息。比如 `C:\Users\Desktop\` 里的文件实际上和桌面里的文件一模一样。
  :::
- **盘符**：能代表特定硬盘的大写英文字母。如路径 `C:\Users\Desktop\file.txt` 中，“C” 就是盘符，代表“此电脑”中的 C 盘。盘符在路径中通常位于最前面，并且后加一个**英文**冒号 `:`。
  :::details
  - 通常情况下盘符都从 C 开始而不是从 A 开始，这是因为在以前，A 和 B 两个盘符是给软盘而非硬盘使用的。
  - 通常情况下操作系统安装在 C 盘，此时 C 盘也叫系统盘。
  - 系统盘**最好使用固态硬盘**而非机械硬盘，这是因为固态硬盘的读写速度往往比机械硬盘快得多，能够给操作系统提供更流畅的使用体验。
  :::
- **扩展名**：详见[文件扩展名](./file-ext-names)。
- **剪贴板**：一个临时区域，用于存储最近一次复制 / 剪切操作的对象。当此对象是文件时，在剪贴板中存储的实际上是文件的路径，而不是文件本身。按下 `Windows` + `V` 打开剪贴板。
- **快捷方式**：详见[文件扩展名](./file-ext-names)。

## 操作

### 1. 选中

使用鼠标单击文件，若文件图标后面出现了淡蓝色背景，则代表该文件被选中。

:::details
- 选中一个文件后，也可以使用键盘的方向键改变选中的文件。
- `Ctrl` + `A` 可以选中当前文件夹下**全部**文件。详见[快捷键](/键盘/hot-keys)。
- 选中一个文件后，按住 `Shift`，再选中另一个文件。这两个文件**之间**的所有文件都会被选中。详见[快捷键](/键盘/hot-keys)。
- 按住 `Ctrl`，可以选中多个文件。详见[快捷键](/键盘/hot-keys)。
:::

选中是操作的前提。

### 2. 打开

使用鼠标**双击**目标文件。

:::tip
选中文件后，按下 `Enter`，这是上文的等效操作。详见[快捷键](/键盘/hot-keys)。
:::

### 3. 剪切

剪切是将文件从原位置移动到剪贴板中的操作。剪切粘贴后，原文件消失。

选中文件后，右键点击，选择“剪切”（可能是图标），若文件图标变为半透明状态，则代表该文件被剪切。

:::tip
选中文件后，按下 `Ctrl` + `X`，这是上文的等效操作。详见[快捷键](/键盘/hot-keys)。
:::

此时可在其他位置粘贴。

:::details
#### 另一种剪切 - 粘贴的方法
打开原文件所在的文件夹和目标文件夹，按住 `Shift`，用鼠标将原文件拖动到目标文件夹。
:::

### 4. 复制

复制是将文件的副本到剪贴板中的操作。复制粘贴后，原文件不受影响。

选中文件后，右键点击，选择“复制”（可能是图标），此时该文件应被复制。

:::tip
选中文件后，按下 `Ctrl` + `C`，这是上文的等效操作。详见[快捷键](/键盘/hot-keys)。
:::

此时可在其他位置粘贴。

:::details
#### 另一种复制 - 粘贴的方法
打开原文件所在的文件夹和目标文件夹，按住 `Shift`，用鼠标将原文件拖动到目标文件夹。
:::

### 5. 粘贴

打开一个文件夹，右键点击空白处，选择“粘贴”（可能是图标）。若弹出了进度条窗口，或文件夹内有新增文件，则代表该文件被粘贴。

:::tip
打开一个文件夹，按下 `Ctrl` + `V`，这是上文的等效操作。详见[快捷键](/键盘/hot-keys)。
:::

### 6. 重命名

选中文件后，右键点击，选择重命名。

:::tip
选中文件后，按下 `F2`（在笔记本电脑上可能是 `Fn` + `F2`），这是上文的等效操作。详见[快捷键](/键盘/hot-keys)。
:::

:::details
#### 另一种重命名的方法
选中文件后，单击文件名。选中和单击这两个动作**要有一定的时间间隔**，这是为了和**双击打开**的动作区分开。
:::

### 7. 创建快捷方式

选中文件后，右键点击，选择“发送到…”，在子菜单选择“桌面快捷方式”。此方法**只能在桌面**创建快捷方式。

:::details
#### 其他创建快捷方式的方法
- 打开原文件所在的文件夹和要创建快捷方式的文件夹，按住 `Ctrl` + `Shift`，用鼠标将原文件拖动到要创建快捷方式的文件夹。
- 选中文件后，按住 `Shift`，右键点击，选择“复制文件路径”。打开要创建快捷方式的文件夹，右键点击空白处，选择“新建”，在子菜单选择“快捷方式”，按照弹出的向导分别填写快捷方式名称和刚刚复制的目标文件路径。
:::
