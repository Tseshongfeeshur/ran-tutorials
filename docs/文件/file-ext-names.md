# 文件扩展名

## 概念

扩展名也叫后缀名，出现在文件名的末尾，以英文句号 `.` 开头。比如在文件 `picture.jpg` 中，`.jpg` 是扩展名。

一般来说，扩展名可以代表文件格式。操作系统在打开文件时，会根据扩展名来决定用什么软件打开。

**可是也有特殊情况**（如扩展名被修改），此时**不能**依赖扩展名判断文件类型。

修改扩展名**不能**改变文件内容，因此修改扩展名**不是**格式转换，修改扩展名本身也**不会**损坏文件。除非在修改扩展名后使用其他软件修改了文件（比如将一个视频文件的扩展名从 `.mp4` 修改为 `.txt` ，然后用记事本打开并保存。这个操作**会导致文件损坏**，即使将扩展名更改回 `.mp4`，视频也无法正常播放）。

一般来说，文件需要有扩展名。不过，没有扩展名的文件也是允许存在的，只是这样可能带来不便。

扩展名**不分大小写**。这意味着 `.JPG`、`.Jpg`、`.jpg` 都代表同一个扩展名。在习惯上，我们**更期望让扩展名全小写**。

## 常用扩展名表

| 分类 | 常见扩展名 | 说明 | 建议打开方式 |
| :--- | :--- | :--- | :--- |
| **文档** | `.txt` | 纯文本。 | Windows 记事本 |
| | `.pdf` | 不可修改，但能保证在多处打开排版不乱。适合打印。可以在其他软件使用“导出”功能，或在打印时选择“打印到 PDF”制作这种文档。 | Google Chrome / Microsoft Edge / Okular |
| | `.md` / `.markdown` | Markdown，语法简单的标记语言，做笔记很好用。 | Windows 记事本 / MarkText |
| **办公** | `.docx` / `.doc` | 使用微软办公套件（Microsoft Office 的 Word）创建的文字文档。`.docx` 更常用更现代。 | Microsoft Office / WPS Office |
| | `.xlsx` / `.xls` | 使用微软办公套件（Microsoft Office 的 Excel）创建的电子表格。常用于统计、数据整理。`.xlsx` 更常用更现代。 | Microsoft Office / WPS Office |
| | `.pptx` / `.ppt` | 使用微软办公套件（Microsoft Office 的 PowerPoint）创建的演示文稿。常用于演讲、汇报。`.pptx` 更常用更现代。 | Microsoft Office / WPS Office |
| **图片** | `.jpg` / `.jpeg` | 最常用。相机拍照时常常使用此格式。有损压缩，因此缺点是不能多次编辑保存，否则画质损失会越来越大。正确的方式是先转换为 `.png` 格式再编辑。 | Windows 照片 / Windows 画图 |
| | `.png` | 无损压缩，支持透明背景，适合保存素材。一般画质比 `.jpg` 图片更好，多次编辑保存画质不会损失。 | Windows 照片 / Windows 画图 |
| | `.gif` | 有损压缩。图片内容可以变化，常用于表情包。缺点是色彩不够逼真。 | Windows 照片 / Google Chrome / Microsoft Edge |
| | `.webp` | 无损 / 有损压缩。较新的图片格式，体积更小，适合在网络上传播。但兼容性稍差。 | Google Chrome / Microsoft Edge |
| | `.bmp` | 无压缩，体积较大。 | Windows 照片 / Windows 画图 |
| | `.heic` | Apple 专有。Windows 需要安装插件或转换后查看，或在从设备导出时就使其先转换格式。实况图片常内含一个短小的视频。 | *<待补充>* |
| | `.psd` | Adobe Photoshop 的工程文件，内部保留了图层及其信息。 | Adobe Photoshop |
| **音频** | `.mp3` | 最常用，兼容性好，音质可能稍差。 | Windows 媒体播放器 / PotPlayer / VLC |
| | `.opus` | 较新的音频格式，压缩算法很好，因此相同码率时通常音质优于 `.mp3`。不过较少见。 | PotPlayer / VLC |
| | `.m4a` | 容器格式，内部可以封装其他格式。手机的录音常常使用此格式。 | Windows 媒体播放器 / PotPlayer / VLC |
| | `.ape` / `.flac` / `.wav` | 无损压缩 / 无压缩。数据相同时一般体积递增（分别是重度压缩、轻度压缩、无压缩）。不过也有例外，因为 `.wav` 有时是容器格式。 | PotPlayer / VLC |
| **视频** | `.mp4` | 最常用，兼容性最好。 | Windows 电影和电视 / PotPlayer / VLC |
| | `.mkv` | 容器格式，经常用于电影，可以封装多个字幕和音频轨道。 | PotPlayer / VLC |
| | `.rmvb` | 以前用于电影。 | PotPlayer / VLC |
| | `.wmv` / `.avi` | 微软早期的格式。 | PotPlayer / VLC |
| **压缩包** | `.zip` | 最常用，兼容性最好。 | Windows 文件资源管理器 / NanaZip |
| | `.7z` | 较常用，更现代，压缩后体积减小得更多。 | NanaZip |
| | `.rar` | 不太常用也不太推荐，压缩为这种格式需要专用软件 WinRAR。 | NanaZip / WinRAR |
| | `.tar` / `.tar.gz` / `.tgz` / `.tar.zst` / `.tar.*` | Linux bsdtar 等制作的打包 / 压缩格式。 | NanaZip |
| | `.apk` | Android 应用安装包，实质上是 ZIP 格式的压缩包。 | Android 软件包管理器 |
| | `.iso` | 光盘镜像文件。Windows 系统安装镜像常以此格式提供。打开需要挂载而非解压。 | Windows 文件资源管理器 |
| **可执行** | `.exe` | Windows 的可执行程序。因为可执行，所以不建议盲目运行陌生 `.exe` 文件，因为不清楚其行为。 | 双击 |
| | `.msi` | 微软标准的安装程序。 | 双击 |
| **网页** | `.html` / `.htm` / `.mhtml` | 网页的源代码或聚合文件。因为其内嵌或引用的 `.js` 脚本文件可执行，所以不建议盲目打开陌生的 `.html` / `.htm` / `.mhtml` 文件，因为不清楚其内部脚本行为。 | Google Chrome / Microsoft Edge |
| **字体** | `.ttf` / `.otf` | 常见的字体文件。使用前需要通过 Windows 字体查看器安装，然后重新启动要使用的软件令其重新加载字体列表。 | Windows 字体查看器 |
| | `.woff` / `.woff2` | 网页字体文件，一般不能直接使用，需要寻找 `.ttf` / `.otf` 版本。 | *<待补充>* |
| | `.fon` | 位图字体文件，现在使用效果不好，建议使用 `.ttf` / `.otf` 格式字体。 | Windows 字体查看器 |
| **快捷方式** | `.lnk` | 一个特殊的小文件，里面保存了目标文件的路径。在打开它时，它会告诉操作系统应该去哪里寻找要打开的文件。一般情况下其图标左下角有一个朝向右上方的小箭头。 | 双击 |
