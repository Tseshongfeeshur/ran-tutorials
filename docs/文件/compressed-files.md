# 压缩文件

## 概念

- **压缩**：一种技术，可以让一堆文件变成一个文件，同时体积缩小。
- **压缩包**：文件经过压缩后得到的成品文件，叫做压缩文件或压缩包。它在网络上传输非常便捷。
- **解压缩**：把压缩文件还原到压缩前的样子。
- 压缩格式：在压缩时使用的方式。不同压缩格式有不同特点，常见的压缩格式有如下几种。详见[文件扩展名](./file-ext-names)。
  - **`.zip`**：最常用，兼容性最好。
  - `.7z`：较常用，更现代，压缩后体积减小得更多。
  - ~~`.rar`：不太常用也不太推荐，压缩为这种格式需要专用软件 WinRAR。~~

:::tip 建议
在日常压缩文件时使用 zip 格式。
:::

## 软件

- 7-zip：好用的压缩 / 解压缩软件。
- **NanaZip**：好用的压缩 / 解压缩软件，由 7-zip 修改而来，优化了外观，与操作系统集成更好。
- ~~WinRAR：每次打开时会弹出订阅提示，只在必须制作 `.rar` 格式压缩包时有用，其他情况不如前两者。~~
- Windows 文件资源管理器：即使不安装第三方软件，现代 Windows 通常也可以良好地处理压缩和解压缩。

:::tip 建议
一般情况下使用 NanaZip 来压缩 / 解压缩。
:::

:::warning 不太建议
使用 WinRAR 来压缩 / 解压缩。它的订阅提示十分烦人，而且并非不可替代。在易用性上也略逊一筹。
:::

:::danger 最好不要
使用其他任何软件来压缩 / 解压缩，软件选择不当会造成包括但不限于广告弹窗、右键菜单污染等一系列复杂的恶性问题。
:::

## 操作

### 压缩

1. 选中所有目标文件
2. 右键单击
3. 在菜单中寻找“压缩到…”
4. *<待补充>*

### 预览压缩包内文件但不解压缩

1. 选中压缩包文件
2. 右键单击
3. 在菜单中寻找“打开方式”
4. 选择“Windows 文件资源管理器”

:::tip 提示
若此方法无效，可能表示 **Windows 文件资源管理器不支持该格式**，此时需要**使用 Nanazip 打开**：

1. 选中压缩包文件
2. 右键单击
3. 在菜单中寻找“Nanazip”
4. 在子菜单中选择“查看压缩包”
:::

### 解压缩

1. 选中压缩包文件
2. 右键单击
3. 在菜单中选择“全部解压缩…”

:::tip 提示
若此方法无效，可能表示 **Windows 文件资源管理器不支持该格式**，此时需要**使用 Nanazip 解压缩**：

1. 选中压缩包文件
2. 右键单击
3. 在菜单中寻找“Nanazip”
4. 在子菜单中选择“解压缩到当前位置”
:::
